# IMDB Runtime Analysis (SQL)

This project analyzes the average runtime of movie genres using the IMDB Top 1000 Movies dataset.

## Business Question
Which genres have the longest average runtime for movies released after 2000?

## SQL Query
See `sql/avg_runtime_by_genre.sql`.

## Result Summary
Fantasy, Action, and Adventure genres have the longest average runtimes.

## Insight
Fantasy films tend to be longer due to world-building and effects-heavy production, which aligns with common film industry patterns.
